Frontend Assignment - Figma to React

You are given a Figma design and asked to convert it into a working React + TypeScript application. The goal is to evaluate your ability to translate design into high-quality UI with realistic interactions.


What you need to do
Implement the UI as close to the Figma design as possible
Add all relevant interaction states (hover, focus, active, loading)
Implement proper error and validation states (e.g. form validation, empty states)
Add subtle, tasteful animations or transitions where appropriate
Deploy the app and share the source code for the same.

Tech stack
React
TypeScript
(Styling, tooling, and animation libraries are up to you.)

How we’ll evaluate
UI accuracy – visual fidelity to Figma (spacing, typography, layout)
Interactions & states – hover, focus, loading, error, and validation states
Code quality – clean component structure and sensible state handling

Submission
Live deployed URL (Vercel, Netlify, etc.)
GitHub repository link
README explaining architecture, decisions and/or enhancements

Best of luck
